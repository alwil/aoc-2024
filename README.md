# Advent of Code 2024

In this repository I try to solve puzzles from 
[Advent of Code 2024](https://adventofcode.com/) in R.


## Project structure 

``` 
aoc2024/
├── R/
│   ├── day01.R
│   ├── day02.R
│   ├── utils.R
├── data/
│   ├── day01_input.csv
│   ├── day02_input.csv
├── renv/
│   ├── (auto-managed environment files)
├── renv.lock
├── aoc2024.Rproj
├── README.md
├── LICENSE
├── .Rprofile
├── .gitignore

```
